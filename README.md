## __VEEBIFOORUM INFO- JA ARVUTITEHNOLOOGIAST "IT VIEW"__
<hr>  
Проект "IT View" – это веб-форум о компьютерных технологиях, предназначенный для обсуждения последних новинок, обмена опытом и решениями проблем в сфере компьютерных технологий. 
Это платформа, где энтузиасты, специалисты и просто интересующиеся могут общаться, получать консультации от экспертов и находить новых единомышленников.

 ### _Технологии  для  __Front-end__ разработки_:
 
Язык программирования – JavaScript , а его программные платформы (framework) – jQuery. Также использовались Bootstrap, CSS. 

 ### _Технологии  для  __Back-end__ разработки_:

Язык программирования – PHP , а его программная платформа (framework) – Laravel.

### База данных - MySQL
1.	XAMPP (дистрибутив для сборки локального веб-сервера)
2.	PhpMyAdmin (веб-интерфейс для управления сайтом и базой данных)

#### Статус: в разработке

## Функциональные возможности:
| Роль | Действие |
| --- | --- |
| _Администратор_ | Может иметь больше прав, чем у приведенных ниже ролей (все права доступа) |
| _Модератор_ | Может получить доступ к админ панели, изменять и удалять данные постов, их комментариев и ответов на комментарии, редактировать данные зарегистрированного пользователя и управлять доступом к аккаунтам,  рассматривать жалобы на зарегистрированных пользователей и управлять доступом к аккаунту|
| _Зарегистрированный пользователь_ | Может зайти в свой аккаунт, создавать новый пост обсуждения, создать новый комментарий в посте обсуждения, создать новый ответ на комментарий, редактировать и удалять свои посты, комментарии и ответы, редактировать свои данные, выйти из аккаунта |
| _Гость_ | Может просматривать всю информацию на сайте, изменить язык сайта, зарегистрироваться |

### Перед тем, как открыть приложение нужно сделать следующие шаги:
- В приложении Github нажать на кнопку "<> Code", далее нажимаем на "Download ZIP". Извлекаем из ZIP папку проекта.
- Смотреть код можно в любом редакторе, на ваше усмотрение (к примеру Visual Studio Code или Sublime).
- Так же нужно будеть открыть дистрибутив для сборки локального веб-сервера XAMPP, если не установлен, то нужно установить https://www.apachefriends.org/download.html
- После запуска XAMPP, нужно запустить Apache и MySQL.
- Чтобы открыть проект, надо извлечь из папки проекта Backup (itview.sql) базы данных "IT View", затем в xampp phpmyadmin создать базу данных itview. В данную базу данных нужно импортировать файл Backup. Для открытия проекта, папку проекта закинуть в папку "htdocs" и после запуска Apache и MySQL в XAMPP, нужно в браузере перейти по адресу http://localhost/название-папки-проекта
- Если, вы не хотите делать последующие действия, то для просмотра можно посетить страницу https://www.itview.site/
