В современном мире компьютерные технологии играют ключевую роль во многих сферах жизни. От развлечений до бизнеса, от образования до научных исследований – компьютеры и интернет проникают во все аспекты нашей жизни. 
В такой динамичной среде актуальной задачей становится создание платформы, где энтузиасты, специалисты и просто интересующиеся могут обсуждать последние новинки, делиться опытом и решениями проблем. 
Веб-форумы становятся важным инструментом обмена знаниями, опытом и идеями в области компьютерных технологий. 
С их помощью пользователи могут обсуждать актуальные темы, делиться решениями проблем, получать консультации от экспертов и находить новых единомышленников.
Именно для решения таких задач создается проект "IT View" – веб-форум о компьютерных технологиях.

_Технологии  для  __Front-end__ разработки_:
 
Язык программирования – JavaScript , а его программные платформы (framework) – jQuery, Bootstrap, CSS. 

_Технологии  для  __Back-end__ разработки_:

Язык программирования – PHP , а его программная платформа (framework) – Laravel.

База данных - MySQL
1.	XAMPP (дистрибутив для сборки локального веб-сервера)
2.	PhpMyAdmin (веб-интерфейс для управления сайтом и базой данных)

__Статус__: в разработке

Функциональные возможности:
| Роль | Действие |
| --- | --- |
| _Администратор_ | Может иметь больше возможностей в управлении зарегистрированными пользователями |
| _Модератор_ | Может получить доступ к админ панели, изменять и удалять данные постов, их комментариев и ответов на комментарии, редактировать данные зарегистрированного пользователя и управлять доступом к аккаунтам,  рассматривать жалобы на зарегистрированных пользователей и управлять доступом к аккаунту|
| _Зарегистрированный пользователь_ | Может зайти в свой аккаунт, создавать новый пост обсуждения, создать новый комментарий в посте обсуждения, создать новый ответ на комментарий, редактировать и удалять свои посты, комментарии и ответы, редактировать свои данные, выйти из аккаунта |
| _Гость_ | Может просматривать всю информацию на сайте, изменить язык сайта, зарегистрироваться |

__Перед тем, как открыть приложение нужно сделать следующие шаги__:
- В приложении Github нажать на кнопку "<> Code", далее нажимаем на "Download ZIP".
- Смотреть код можно в любом редакторе, на ваше усмотрение (к примеру Visual Studio Code или Sublime).
- Так же нужно будеть открыть дистрибутив для сборки локального веб-сервера XAMMP, если не установлен, то нужно установить https://www.apachefriends.org/download.html
- Чтобы открыть проект , надо достать из ZIP файла Backup базы данных "IT View". Папку проекта закинуть в папку "htdocs" и после запуска apache, xammp нужно в браузере написать https://localhost/название-папки-проекта
